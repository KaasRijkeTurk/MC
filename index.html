● ● ●
microslop-support-bot@system:~
── utility commands ──
.help — toont alle beschikbare parameters
.uptime — actieve server responstijd
.purge [1-100 / all] [ww] — database en chat opschoning
.poll [vraag] | [opt1] | [opt2] — genereer dynamische data-polls
.userinfo [@user] — fetch metadata van gespecificeerde client
.serverinfo — output server metrics en statistieken
── server beheer ──
.setup [ww] — automatische initialisatie (kanalen, rollen, perms)
.setwelcome #kanaal — bind webhook aan specifiek kanaal
.setautorole @rol — configureer on-join role assignments
── minecraft integratie ──
.mcstatus — ping de server API voor actuele data
.mcplayers — lijst van geverifieerde actieve sessies
.mcinfo — uitgebreide server en backend logs
.mcrcon [commando] [ww] — execute RCON commando op afstand
